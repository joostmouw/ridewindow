class RideWindowApp extends ConsumerWidget {
  const RideWindowApp({super.key});

**_buildDayChip(DateTime day, SlotsState slotsState):**
- Bepaal klasse (bad/ok/good) op basis van beste tier van slots op die dag:
  - SlotsLoaded met >= 1 slot van dag: Perfect/Great → 'good' (groen); Acceptable → 'ok' (amber); geen slots → 'bad' (rood)
  - Nog aan het laden → 'bad' (grijs skeleton — zie loading state hieronder)
- Kleur per klasse: good → #E8F5E9 + #2E7D32; ok → #FFF3E0 + #E65100; bad → #FFEBEE + #C62828
- selected: extra ring `BoxDecoration(borderRadius: circular(50), border: Border.all(color: greenColor, width: 2.5))`
- Tap: `setState(() { _selectedDay = _selectedDay?.day == day.day ? null : day; })`
- GestureDetector(onTap, child: Column(Text(dayLabel), SizedBox(8), Container(34x34, cirkel, kleur + text)))
- dayLabel: 'MA'/'DI'/'WO'/'DO'/'VR'/'ZA'/'ZO'

**_buildCardsSection(AsyncValue<List<HourlyForecast>> weatherState, SlotsState slotsState):**
Drie gevallen:
1. **weatherState.isLoading:** toon _buildSkeletonCards()
2. **SlotsLoaded met reason != null of slots.isEmpty:**
   - reason == SlotsEmptyReason.badWeather → "Geen goede rijmomenten deze week. Slecht weer verwacht."
   - reason == SlotsEmptyReason.allBlocked → "Alle goede momenten zijn geblokkeerd. Pas je schema aan."
   - reason == null maar lege lijst → "Geen rijmomenten gevonden."
   - Toon als Center(Padding(child: Column(icon + text)))
3. **SlotsLoaded met slots:**
   - Filter als _selectedDay != null: `slots.where((s) => s.start.day == _selectedDay!.day).toList()`
   - Sorteer op tier (Perfect eerst)
   - ListView.builder met _buildRideCard(slot)
