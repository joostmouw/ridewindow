## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| ProfileScreen → GPS permissions system | Gebruiker kan toestemming weigeren; app moet dit graceful afhandelen |
| WeatherNotifier → Open-Meteo API | Locatiecoördinaten worden verstuurd naar externe API |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-03-01 | Information Disclosure | GPS coördinaten naar Open-Meteo | accept | Open-Meteo is free, no-key service; coördinaten zijn noodzakelijk voor het kernfunctioneren; privacy policy vermeldt locatiegebruik (Phase 10) |
| T-07-03-02 | Tampering | City name in setLocationOverride() | accept | Waarde komt uitsluitend uit de gehardcodeerde kNlCities lijst in de stad-picker; gebruiker kan geen willekeurige string invoeren |
| T-07-03-03 | Denial of Service | WeatherNotifier opnieuw ophalen bij elke locatiewijziging | accept | Riverpod's auto-dispose + caching beperkt onnodige API-aanroepen; WeatherRepository cache policy (Phase 2) hergebruikt data binnen 1 uur |
</threat_model>
