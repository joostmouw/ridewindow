**Constructor:** accepteert optionele `FlutterLocalNotificationsPlugin? plugin` parameter voor testbaarheid (DI).

**`Future<void> init()` methode:**
1. Initialiseer plugin met `AndroidInitializationSettings('@mipmap/ic_launcher')`
2. Maak beide kanalen aan via `resolvePlatformSpecificImplementation<AndroidFlutterLocalNotificationsPlugin>()`

**`Future<bool> requestPostNotificationsPermission()` (per NOTIF-04, D-08-08):**
- Gebruik `Permission.notification.request()` van `permission_handler`
- Retourneer `status.isGranted`

**`Future<bool> canScheduleExact()` (per NOTIF-05, D-08-09):**
- Gebruik `androidPlugin?.canScheduleExactNotifications() ?? false`
- Retourneer het resultaat

**`Future<void> openExactAlarmSettings()` (per NOTIF-05):**
- Roep `androidPlugin?.requestExactAlarmsPermission()` aan
- Val terug op `openAppSettings()` van permission_handler als dat faalt

**`Future<void> scheduleEveningBefore({required DateTime slotDay, required String slotTitle, required bool exact})` (per NOTIF-01):**
- Bereken `scheduledDate = TZDateTime(tz.local, slotDay.year, slotDay.month, slotDay.day - 1, 19, 0)` — 19:00 de VORIGE dag
- Sla over als scheduledDate in het verleden ligt (`scheduledDate.isBefore(TZDateTime.now(tz.local))`)
- Gebruik `androidScheduleMode: exact ? AndroidScheduleMode.exactAllowWhileIdle : AndroidScheduleMode.inexact`
- Notificatie-ID: `kNotifIdEveningBefore`
- Titel: `'Top rijmoment morgen!'`, body: `'$slotTitle — perfecte omstandigheden verwacht'`

**`Future<void> scheduleMorningOf({required DateTime slotStart, required String slotTitle, required bool exact})` (per NOTIF-02):**
- Bereken `scheduledDate = TZDateTime.from(slotStart.subtract(const Duration(hours: 2)), tz.local)`
- Sla over als scheduledDate in het verleden ligt
- Notificatie-ID: `kNotifIdMorningOf`
- Titel: `'Over 2 uur een top rijmoment!'`, body: `'$slotTitle — maak je klaar om te rijden'`

**`Future<void> scheduleWeeklyDigest({required String bodySummary, required bool exact})` (per NOTIF-03):**
- Bereken eerstvolgende zondag 19:00 in lokale timezone:
  ```dart
  final now = TZDateTime.now(tz.local);
  final daysUntilSunday = (DateTime.sunday - now.weekday + 7) % 7;
  final nextSunday = TZDateTime(tz.local, now.year, now.month, now.day + (daysUntilSunday == 0 ? 7 : daysUntilSunday), 19, 0);
  ```
- Kanaal: `'weekly_digest'`
- Notificatie-ID: `kNotifIdWeeklyDigest`
- Titel: `'Je rijoverzicht voor deze week'`, body: bodySummary

**`Future<void> cancelAll()`:** roep `_plugin.cancelAll()` aan.
