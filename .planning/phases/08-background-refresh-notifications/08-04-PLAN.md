**Hulpmethode `_scheduleNotificationsIfPermitted()`** (per NOTIF-04, NOTIF-05):
```dart
Future<void> _scheduleNotificationsIfPermitted(BuildContext context) async {
  // 1. Vraag POST_NOTIFICATIONS op
  final granted = await _notifService.requestPostNotificationsPermission();
  if (!granted) return;

  // 2. Controleer SCHEDULE_EXACT_ALARM
  final canExact = await _notifService.canScheduleExact();
  if (!canExact) {
    // Toon SnackBar met uitleg (per NOTIF-05 fallback, D-08-09)
    if (context.mounted) {
      ScaffoldMessenger.of(context).showSnackBar(
        SnackBar(
          content: const Text(
            'Exacte timing niet gegarandeerd. Sta exacte alarmen toe in Instellingen voor betrouwbaarheid.',
          ),
          action: SnackBarAction(
            label: 'Instellingen',
            onPressed: () => _notifService.openExactAlarmSettings(),
          ),
          duration: const Duration(seconds: 6),
        ),
      );
    }
  }
  // Verdere scheduling vindt plaats via SlotsNotifier data in de toekomst (Phase 8 scope: permissie-flow)
}
```

**Voeg WidgetsBindingObserver toe aan _HomeScreenState** (per D-08-13, NOTIF-06):
1. Voeg `with WidgetsBindingObserver` toe aan de klasse-declaratie (naast het bestaande `SingleTickerProviderStateMixin`)
2. In `initState()`: voeg `WidgetsBinding.instance.addObserver(this);` toe
3. In `dispose()`: voeg `WidgetsBinding.instance.removeObserver(this);` toe vóór `super.dispose()`
4. Voeg `didChangeAppLifecycleState()` override toe:
```dart
@override
void didChangeAppLifecycleState(AppLifecycleState state) {
  if (state == AppLifecycleState.resumed) {
    // Herlaad lastRefreshed timestamp bij terugkeer naar foreground (NOTIF-06)
    ref.read(lastRefreshedProvider.notifier).refresh();
  }
}
```

## Vertrouwensgrenzen

| Grens | Beschrijving |
|-------|-------------|
| SwitchListTile onChanged → async gap | BuildContext kan invalid worden na await; `context.mounted` check vereist |
| WidgetsBindingObserver → lifecycle | Observer moet verwijderd worden in dispose() om memory leaks te voorkomen |

| Bedreiging-ID | Categorie | Component | Dispositie | Mitigatieplan |
|--------------|-----------|-----------|------------|---------------|
| T-08-04-01 | Denial of Service | async context.mounted niet gecheckt | mitigate | Elke onChanged heeft `if (v && context.mounted)` guard vóór async calls |
| T-08-04-02 | Denial of Service | WidgetsBindingObserver niet verwijderd | mitigate | `removeObserver(this)` in dispose() staat expliciet in de taakinstructie |
